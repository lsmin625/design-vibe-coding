# 액터별 화면 목록 - 일반 고객

## 프롬프트

OSS 패러디 디자인 플랫폼의 일반 고객은 다음과 같은 유즈케이스 목록을 포함한다.

- 회원가입 및 로그인: 이메일 또는 SNS 계정을 통한 회원가입·로그인
- 상품 탐색 및 검색: 카테고리, 키워드, 인기순/신상품 정렬 검색
- 디자인 상세보기: 디자이너 정보, 제작 설명, 기부 비율 확인
- 상품 주문: 옵션 선택(색상, 사이즈 등), 장바구니 및 결제
- 주문/배송 조회: 주문 내역 및 배송 현황 실시간 조회
- 리뷰 및 SNS 공유: 구매 후 상품 후기 작성 및 소셜 공유

유즈케이스 목록을 기준으로 **화면 목록을 도출**하여 표로 정리한다. **화면은 Vue 라우터 기준 경로 단위로 구분되는 UI 컴포넌트를 의미**한다.
예를 들어, 회원가입 및 로그인 유즈케이스는 회원가입 화면과 로그인 화면으로 분리 되어야 한다. 

화면 목록 표의 컬럼 : 화면명(한글), 화면 컴포넌트(영문-Pascal Case), 입력 (검색 조건, 필터 조건, 조회 기간 및 CRUD 버튼), 출력 데이터(그리드에 포함될 컬럼 항목 등), 연동 화면
- 연동 화면은 로그인 화면에서 이동할 다음 경로로 상품 탐색 및 검색이 메인 화면인 경우 이에 해당한다. 회원가입-로그인 화면은 서로 연동될 수 있다.
